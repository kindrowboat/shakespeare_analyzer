# ShakespeareAnalyzer

A test assignment project for ThoughtBot TDD.

# Running

    git clone https://github.com/aauthor/shakespeare_analyzer.git
    cd shakespeare_analyzer
    bundle install
    ruby bin/macbeth_analyzer.rb
    
# Expected Output

    $ ruby bin/macbeth_analyzer.rb
    62 First Witch
    27 Second Witch
    27 Third Witch
    ...
